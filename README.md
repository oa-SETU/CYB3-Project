# CYB 3rd Year Project Submission
Clone this repository and follow the folder structure to push your project code and report.

```
📦 CYB3-Project
├─ 📂Guidelines
│  └─ 🗎 FinalReports-Guidelines.pdf
├─ 📂Project Code
├─ 📂Reports
└─ 🗎 README.MD

```
The Guidelines folder already contains a guideline document to assist you with your end of project documentation. 

Please ensure that your `Project Code` and `Reports` are in their respective folders.
